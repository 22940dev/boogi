:wave: Hello!

This is a [GatsbyJS](http://gatsbyjs.org/) documentation template. GatsbyJS
allows generating beautiful sites from simple _Markdown_ pages

This template is based on popular [Gitbook](https://gitbook.com) look and feel.
It offers custom styling and components that would enable building beautiful
documentation for projects and products quickly.
